# pulsarai

> Your Ecommerce Shopping Assistant

## Libraries used
numpy
pandas
sklearn
spacy
nltk

## How To Use


### Setup
```
source venv/bin/activate
export FLASK_APP=index
sudo apt-get install python3-pip
sudo apt-get install python3-flask
pip install scikit-learn
flask run
```

### Run The App

```
flask run
```
